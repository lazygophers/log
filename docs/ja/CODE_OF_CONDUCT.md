# 📋 行動規範

## 私たちのコミットメント

メンバー、貢献者、そしてリーダーとして、私たちは年齢、体型、可視または不可視の障害、人種、ジェンダー表現、ジェンダーアイデンティティと表現、経験レベル、教育、経済的社会的地位、国籍、身体的特徴、人種、キャスト、肌の色、宗教、性指向やアイデンティティに関係なく、すべての人に対して嫌がらせのないコミュニティ参加体験を作り出すことを約束します。

私たちは、開放的で歓迎的で多様で包摂的で健全なコミュニティを促進する方法で行動し、対話することを約束します。

## 私たちの基準

積極的な環境を作るための行動の例には以下が含まれます：

### ✅ 鼓励される行動

-   **他者に対して共感と善意を示す**
-   **異なる意見、視点、経験を尊重する**
-   **建設的なフィードバックを与え、受け入れることを優雅に行う**
-   **責任を取り、私たちの過ちの影響を受けた人に謝罪し、経験から学ぶ**
-   **全体のコミュニティにとって最善であることに焦点を当て、個人の利益だけでなく**
-   **歓迎的で包摂的な言語を使用する**
-   **貢献に感謝し、貢献の大小に関わらず**
-   **新人をサポートし、歓迎感を持たせる**

### ❌ 不適切な行動

不適切な行動の例には以下が含まれます：

-   **嫌がらせ**: 性的暗示のある言葉や画像、およびあらゆる形態の性的な注目や接近
-   **挑発**: 侮辱的または貶めるようなコメント、および個人や政治的な攻撃
-   **差別**: 保護されたグループを対象とする差別的な発言や行為
-   **プライバシー侵害**: 物理的な住所やメールアドレスなどの他人の個人情報を明示的な許可なく公開する
-   **スパム**: 過度に大量または無関係な投稿、または商業広告
-   **無関係なコンテンツ**: プロジェクトに関係のないコンテンツを継続的に投稿する
-   **その他の行動**: プロフェッショナルな環境で不適切と考えられるその他の行為

## 実施責任

コミュニティリーダーは、私たちの許容される行動基準を明確にし、実施する責任を負い、不適切、脅威的、冒涜的、または有害と見なされる行動に対して、適切かつ公平な是正措置を講じることを約束します。

コミュニティリーダーは、この行動規範に準拠しないコメント、提出、コード、ウィキ編集、問題、その他の貢献を削除、編集、または拒否する権利と責任を持ち、適切な場合は審査決定の理由を共有します。

## 適用範囲

この行動規範は、すべてのコミュニティの空間に適用され、個人が公共の空間で正式にコミュニティを代表する場合にも適用されます。私たちのコミュニティを代表する例には、公式のメールアドレスの使用、公式なソーシャルメディアアカウントを通じたコンテンツの投稿、オンラインまたはオフラインのイベントでの指定代表の務めが含まれます。

### コミュニティ空間には以下が含まれます

-   **GitHub リポジトリ**およびすべての関連する空間（問題、プルリクエスト、ディスカッション、ウィキ）
-   **公式なコミュニケーションチャネル**（Discord、Slack、メーリングリスト）
-   **プロジェクトのソーシャルメディア**アカウントと投稿
-   **プロジェクトのイベント**（会議、集まり、ワークショップ）
-   **ドキュメント**およびその他の公式なプロジェクト資料

## 実施メカニズム

### 報告

:::info 通報チャネル

悪用、嫌がらせ、またはその他の許容できない行為のインスタンスは、実施を担当するコミュニティリーダーに報告できます：

-   **主要な連絡先**: support@lazygophers.com
-   **GitHub Issues**: 公開の違反行為については、GitHub Issues を通じて報告できます
-   **秘密の通報**: 敏感な事項については、直接メールでお問い合わせください

すべての苦情は、タイムリーかつ公平に審査および調査されます。すべてのコミュニティリーダーは、いかなる事件の通報者のプライバシーと安全も尊重する義務を負います。

:::

### 応答タイムライン

-   **確認**: 24-48時間以内
-   **初期応答**: 72時間以内
-   **解決**: 1-2週間以内（複雑なケースにはより長い時間がかかる場合があります）

## 実施ガイドライン

コミュニティリーダーは、違反の結果を決定するために、これらのコミュニティ影響ガイドラインに従います：

### 1. 修正

:::warning 初回の違反

**コミュニティへの影響**: 社区内で不適切と見なされる言語や、プロフェッショナルではないまたは歓迎されない行為の使用。

**結果**: コミュニティリーダーからの秘密の書面での警告、違反の性質を明確に説明し、なぜその行為が不適切であるかを説明します。公開の謝罪を求められる場合があります。

**措置の例**：

-   問題を秘密に説明する
-   違反内容の編集または削除を要求する
-   特定のプロジェクト領域での活動を一時的に制限する

:::

### 2. 警告

:::warning 重大または継続的な違反

**コミュニティへの影響**: 単一のイベントまたは一連の行動違反。

**結果**: 行動が続いた場合に結果が生じることを明記した警告。指定された期間内に、関連する人物とやり取りしないこと、実施者を含む請求されないやり取りも含めます。これは、コミュニティ空間および外部チャネル（ソーシャルメディアなど）でのやり取りを避けることを意味します。これらの条項の違反は、一時的または永久的な禁止につながる可能性があります。

**措置の例**：

-   書面での警告、行動の期待を明確にする
-   コミュニティ活動への参加を一時的に制限する
-   その後の貢献の監視

:::

### 3. 一時的な禁止

:::danger 一時的な禁止

**コミュニティへの影響**: コミュニティの基準の重大な違反、継続的な不適切な行動を含む。

**結果**: 指定された期間、コミュニティとのあらゆるやり取りやあらゆる形態の公開コミュニケーションを一時的に禁止します。この期間中、関連する人物と公開または非公開でやり取りすることは許可されません、請求されないやり取りを含めます。これらの条項の違反は、永久的な禁止につながる可能性があります。

**継続時間**: 通常1-3ヶ月、深刻度に応じて異なります

:::

### 4. 永久的な禁止

:::danger 永久的な禁止

**コミュニティへの影響**: コミュニティの基準の違反パターンを示し、継続的な不適切な行為、個人への嫌がらせ、または個人への攻撃または貶めを含む。

**結果**: コミュニティ内のあらゆる形態の公開やり取りの永久的な禁止。

:::

## 申立プロセス

不当な処罰を受けたと考えられる場合は、以下の方法で申立てを行うことができます：

1. **メールで送信**: 30日以内に support@lazygophers.com 宛て
2. **提出**: 申立の詳細な理由を提供する
3. **添付**: 関連する証拠または背景情報を添付する
4. **待機**: 別のコミュニティリーダーグループによる審査を待つ

申立は2-3週間以内に審査され、その時点で行われる決定は最終的な決定となります。

## 積極的なコミュニティの構築

:::tip 良好的なコミュニティを共同で作りましょう

私たちはコミュニティメンバーを奨励します：

### 🤝 包摂の促進

-   **新人を歓迎し、有益なガイダンスを提供する**
-   **多様性を尊重し、異なる背景と視点を包摂する**
-   **すべてのコミュニケーションで包摂的な言語を使用する**
-   **学習中や質問している人に忍耐を持つ**

### 🎯 解決策への焦点

-   **批評だけでなく、建設的なフィードバックを提供する**
-   **誰かが困っているのを見たときに積極的に助けを提供する**
-   **知識とリソースを寛大に共有する**
-   **競争ではなく協力する**

### 🌟 貢献を認める

-   **貢献の大小に関わらず、貢献者に感謝する**
-   **コミュニティメンバーの成果を称賛する**
-   **他人が学び成長するのを指導する**
-   **他人のアイデアや作業を適切に帰属させる**

:::

## 技術基準

行動基準に加えて、以下の技術基準も維持しています：

### コード品質

-   既存のコーディング標準に従う
-   明確で、適切に文書化されたコードを記述する
-   貢献を十分にテストする
-   コードレビューのフィードバックを心に留める

### コミュニケーション規範

-   議論がプロジェクトに関連していることを維持する
-   問題とプルリクエストで明確でプロフェッショナルな言語を使用する
-   エラーの報告や機能リクエスト時に役立つコンテキスト情報を提供する
-   メンテナーの返信を待つ間、忍耐を持つ
