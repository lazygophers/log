# LazyGophers Log

一個全面的 Go 語言日誌庫，提供靈活的日誌記錄功能。

## 特性

- 支援多種日誌級別
- 靈活的輸出格式配置
- 多目的地輸出
- 性能優化
- 易於擴展

## 快速開始

```go
import "github.com/lazygophers/log"

func main() {
    log.Info("你好，世界！")
    log.Error("發生錯誤", "error", err)
}
```
