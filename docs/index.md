# LazyGophers Log

一个全面的 Go 语言日志库，提供灵活的日志记录功能。

## 特性

- 支持多种日志级别
- 灵活的输出格式配置
- 多目的地输出
- 性能优化
- 易于扩展

## 快速开始

```go
import "github.com/lazygophers/log"

func main() {
    log.Info("Hello, World!")
    log.Error("An error occurred", "error", err)
}
```
