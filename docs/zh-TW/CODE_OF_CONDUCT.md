# 📋 行为准则

## 我们的承诺

作为成员、贡献者和领导者，我们承诺为每个人创造一个无骚扰的社区参与体验，无论年龄、体型、可见或不可见的残疾、种族、性别特征、性别认同和表达、经验水平、教育、社会经济地位、国籍、个人外貌、种族、种姓、肤色、宗教或性取向和身份如何。

我们承诺以促进开放、欢迎、多元、包容和健康社区的方式行动和互动。

## 我们的标准

为社区创造积极环境的行为示例包括：

### ✅ 鼓励的行为

-   **对他人表现出同理心和善意**
-   **尊重不同的意见、观点和经验**
-   **给予并优雅接受建设性反馈**
-   **承担责任**并向受我们错误影响的人道歉，并从经验中学习
-   **关注对整体社区最有利的事情**，而不仅仅是个人的利益
-   **使用欢迎和包容的语言**
-   **对贡献表示感谢**，无论贡献大小
-   **帮助新人**入门并感到受欢迎

### ❌ 不可接受的行为

不可接受的行为示例包括：

-   **骚扰**：使用性暗示语言或图像，以及任何形式的性关注或接近
-   **挑衅**：侮辱性或贬低性评论，以及个人或政治攻击
-   **歧视**：针对任何受保护群体的歧视性言论或行为
-   **隐私侵犯**：未经明确许可发布他人的私人信息，如物理地址或电子邮件地址
-   **垃圾信息**：过度或无关的发布，或商业广告
-   **无关内容**：持续发布与项目无关的内容
-   **其他行为**：在专业环境中可能被认为不合适的其他行为

## 执行责任

社区领导者负责澄清和执行我们的可接受行为标准，并将对任何他们认为不适当、威胁性、冒犯性或有害的行为采取适当和公平的纠正措施。

社区领导者有权并有责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、维基编辑、问题和其他贡献，并在适当时沟通审核决策的原因。

## 适用范围

本行为准则适用于所有社区空间，也适用于个人在公共空间中正式代表社区时。代表我们社区的示例包括使用官方电子邮件地址、通过官方社交媒体账户发布内容，或在线上或线下活动中担任指定代表。

### 社区空间包括

-   **GitHub 仓库**及所有相关空间（问题、拉取请求、讨论、维基）
-   **官方沟通渠道**（Discord、Slack、邮件列表）
-   **项目社交媒体**账户和帖子
-   **项目活动**（会议、聚会、研讨会）
-   **文档**及其他官方项目材料

## 执行机制

### 报告

:::info 举报渠道

滥用、骚扰或其他不可接受行为的实例可以向负责执行的社区领导者报告：

-   **主要联系方式**：support@lazygophers.com
-   **GitHub Issues**：对于公开的违规行为，您可以通过 GitHub Issues 报告
-   **私密报告**：对于敏感事项，请通过电子邮件直接联系我们

所有投诉都将得到及时和公平的审查和调查。所有社区领导者都有义务尊重任何事件报告者的隐私和安全。

:::

### 响应时间表

-   **确认**：24-48 小时内
-   **初步回应**：72 小时内
-   **解决**：1-2 周内（复杂案例可能需要更长时间）

## 执行指南

社区领导者将根据社区影响矩阵遵循这些社区影响指南来确定违规行为的后果：

### 1. 纠正

:::warning 第一次违规

**社区影响**：使用不恰当的语言或其他在社区中被认为不专业或不受欢迎的行为。

**后果**：社区领导者的私人书面警告，明确说明违规行为的性质，并解释为什么该行为不恰当。可能会要求公开道歉。

**示例措施**：

-   私下说明问题
-   要求编辑或删除违规内容
-   暂时限制在特定项目区域的活动

:::

### 2. 警告

:::warning 严重或持续违规

**社区影响**：通过单一事件或一系列行为违规。

**后果**：警告，说明如果行为持续将产生后果。在指定时间内不得与相关人员互动，包括与执行行为准则的人员进行未经请求的互动。这包括避免在社区空间以及外部渠道（如社交媒体）中的互动。违反这些条款可能会导致临时或永久禁令。

**示例措施**：

-   书面警告并明确行为预期
-   暂时限制参与社区活动
-   监控后续贡献

:::

### 3. 临时禁令

:::danger 临时禁令

**社区影响**：严重违反社区标准，包括持续不适当行为。

**后果**：在指定时间内暂时禁止与社区的任何互动或任何形式的公共沟通。在此期间，不允许与相关人员公开或私下互动，包括与执行行为准则的人员进行未经请求的互动。违反这些条款可能会导致永久禁令。

**持续时间**：通常为 1-3 个月，具体取决于严重程度

:::

### 4. 永久禁令

:::danger 永久禁令

**社区影响**：表现出违反社区标准的模式，包括持续不适当行为、骚扰个人或对个人进行攻击或贬低。

**后果**：永久禁止在社区内进行任何形式的公共互动。

:::

## 申诉流程

如果您认为自己受到了不正确的处罚，您可以通过以下方式提出申诉：

1. **发送电子邮件**至 support@lazygophers.com，在 30 天内提出
2. **提供**申诉的详细理由
3. **附上**任何相关的证据或背景信息
4. **等待**由另一组社区领导者进行审查

申诉将在 2-3 周内审查完毕，届时做出的决定为最终决定。

## 建设积极社区

:::tip 共建美好社区

我们鼓励社区成员：

### 🤝 促进包容

-   **欢迎新人**并提供有益的指导
-   **尊重多样性**，包容不同的背景和观点
-   **使用包容性语言**进行所有沟通
-   **对正在学习或提问的人保持耐心**

### 🎯 聚焦解决方案

-   **提供建设性反馈**，而不仅仅是批评
-   **主动提供帮助**，当看到有人遇到困难时
-   **慷慨分享知识和资源**
-   **协作而非竞争**

### 🌟 认可贡献

-   **感谢贡献者**的工作，无论贡献大小
-   **表彰社区成员**的成就
-   **指导他人**学习和成长
-   **正确归属他人的想法和工作**

:::

## 技术标准

除行为标准外，我们还维护以下技术标准：

### 代码质量

-   遵循既定的编码规范
-   编写清晰、文档完善的代码
-   充分测试您的贡献
-   虚心接受代码审查反馈

### 沟通规范

-   保持讨论与项目相关
-   在问题和拉取请求中使用清晰、专业的语言
-   在报告错误或请求功能时提供有用的上下文信息
-   在等待维护者回复时保持耐心
